# 探秘人工智能 - AI 入门课程

面向小学生的互动式 AI 科普课程，帮助孩子们理解人工智能的基本原理和应用。

## 📋 项目简介

这是一个基于 Next.js 开发的互动式幻灯片演示系统，专为小学生设计的人工智能入门课程。通过游戏化、可视化的方式，让抽象的 AI 概念变得生动有趣。

### 核心特点

- 🎮 **互动体验**：声音投票、画板绘画、点击揭秘等多种互动方式
- 🎨 **精美设计**：使用 Tailwind CSS 打造的现代化 UI
- 📱 **响应式布局**：适配各种屏幕尺寸
- 🎬 **多媒体支持**：视频、音频、动画效果
- 🔗 **外部链接**：集成 Quick Draw、AI 训练平台等外部资源

## 🛠️ 技术栈

- **框架**: Next.js 15 (App Router)
- **语言**: TypeScript
- **样式**: Tailwind CSS
- **UI 组件**: Lucide React Icons
- **部署**: Vercel

## 🚀 快速开始

```bash

# 启动开发服务器
npm run dev

# 打开浏览器访问
# http://localhost:3000
```

### 构建

```bash
# 生产环境构建
npm run build

# 启动生产服务器
npm start
```

## 🎯 互动功能说明

### P8 - 神经网络声音投票
- 使用麦克风收集声音
- 实时分贝仪显示
- 梯度敏感度设计（80dB 达到最大值）
- 手动 +/- 按钮作为备用

### P11 - 你画我猜画板
- Canvas 绘图功能
- 支持鼠标绘画
- 一键清除功能
- 跳转链接到 Quick Draw

### P12 - 训练自己的 AI
- 介绍 AI 训练三步流程
- 跳转链接到 train.aimaker.space

## 🌐 部署到 Vercel

### 方法一：通过 Vercel CLI

```bash
# 安装 Vercel CLI
npm i -g vercel

# 登录 Vercel
vercel login

# 部署
vercel
```

### 方法二：通过 Vercel 网站

1. 将代码推送到 GitHub
2. 访问 [vercel.com](https://vercel.com)
3. 点击 "Import Project"
4. 选择你的 GitHub 仓库
5. 点击 "Deploy"

本项目无需额外环境变量配置。
